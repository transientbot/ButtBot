Idle Settings

Enable Chat Idleness

Set the amount of time that someone can be silent in chat and still receive channel currency. Set these to 0 to allow everyone in chat to receive currency regardless of activity.

Set the number of points that a user will receive when they have been silent for the above duration. Setting a value higher than your points per interval will result in idlers receiving more points than active chatters, and setting a value lower than 0 will penalize users points every interval. Points may drop below 0. This value will only be awarded while you are live; while you are offline, all users will receive your full offline points value.

Set the amount of time that someone can be silent in chat and still be targeted by (random) commands. Set these to 0 to allow everyone in chat to be targeted by (random) commands regardless of activity.
